# Student Exam Preparation Project

This project is designed to help students from classes 6 to 12 prepare for their exams through interactive learning and practice. The platform includes various subjects and utilizes games to make learning engaging and fun.

## Features

- Subjects for classes 6 to 12
- Interactive quizzes and games
- Responsive design using HTML, CSS, and JavaScript

## Project Structure

- `src/index.html`: Main entry point of the application.
- `src/css/styles.css`: Styles for the application.
- `src/js/app.js`: JavaScript functionality for the application.
- `src/subjects/`: Contains subject-specific HTML files for each class.
- `src/data/`: Contains quiz data and syllabus information in JSON format.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Open `index.html` in your web browser to start using the application.

## Usage

Navigate through the subjects and take quizzes to test your knowledge. Enjoy learning!
